# 公開運用メモ

このフォルダには **外部共有可能な完成記事だけ** を配置する。

## Public移行ルール

1. 記事の `status` を `verified` または `published` にする
2. `confidence` が `medium` 以上であることを確認する
3. `70_AI生成用素材/プロンプト/公開前レビュープロンプト` でレビューする
4. 問題なければ `80_Public` 配下の適切なサブフォルダへ複製または移動する
5. GitHub Pages公開時は `docs/` へエクスポートする

## 推奨サブフォルダ構成

```text
80_Public/
├── tactics/
├── maps/
├── weapons/
├── case-studies/
└── assets/
```

## 公開不可情報（このフォルダに置かない）

- フレンド名・個人名
- 個別プレイログの詳細
- 未検証の断定的戦術
- 非公開にしたいルートや運用メモ
- `02_Private` の内容

## 公開判定チェックリスト

- [ ] 個人名・フレンド名がない
- [ ] 未確認事項が断定されていない
- [ ] 画像に非公開情報が含まれていない
- [ ] タグがタグ辞書に沿っている
- [ ] 関連記事リンクがある

## GitHub Pages

`80_Public` 配下の記事を `docs/` へ反映する。詳細は `GitHubPages_移行設計.md` を参照。
