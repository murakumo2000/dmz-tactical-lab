# 公開前チェックリスト

`80_Public` から `docs/` へ同期する前に確認する。

## 必須チェック

- [ ] 内部パスが含まれていない
- [ ] 個人名・フレンド名が含まれていない
- [ ] 未確認情報を断定していない
- [ ] title が設定されている
- [ ] description が設定されている
- [ ] status が設定されている
- [ ] confidence が設定されている
- [ ] updated が設定されている
- [ ] 関連記事リンクが存在するページを指している
- [ ] トップまたは一覧への戻りリンクがある

## 推奨チェック

- [ ] 記事冒頭だけで内容が分かる
- [ ] 適用条件が明確
- [ ] DMZ戦術研究所らしい検証観点がある
- [ ] 画像・図解に非公開情報が含まれていない
