# 岩場戦闘の基本原則

## 原則

1. 高所を取られている場合は長居しない
2. 射線が通る稜線上で停止しない
3. 退路を確保してから撃つ
4. 敵を倒すより、敵の位置を確定させることを優先する

## 判断の要点

| 状況 | 判断 |
|---|---|
| 敵が上を取っている | 迂回または撤退 |
| AIが反応している | 第三者への露見を警戒 |
| 脱出時間が近い | 戦闘継続しない |
