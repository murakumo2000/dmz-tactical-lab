# 建物戦の基本原則

## 原則

1. 入口は1つに限定し、別ルートの存在を想定する
2. フロアを上げるたびに降りるルートを確認する
3. 1フロアの滞在は短時間に留める
4. 撃ち合いが膠着したら降りて離脱する

## 判断の要点

- 敵位置不明での突入は避ける
- 長時間撃ち合い → 第三者到着リスク → 離脱
