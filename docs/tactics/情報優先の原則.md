# 情報優先の原則

## 情報の優先順位

1. 敵の人数（単数か複数か）
2. 敵の方向（1方向か複数か）
3. 第三者の存在（銃声・足音）
4. 退路の確保状況

## 実践

初動30秒はピークと音の収集に使う。不明なまま突入しない。
