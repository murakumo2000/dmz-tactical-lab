# アシカ島の基本行動

## 基本原則

1. スポーン直後は音と視線を抑えて情報収集する
2. 海岸線の直線移動は短時間に留める
3. 縦移動中は側面からの挟撃を常に想定する
4. 脱出時間を意識し、マップ端での長時間滞在を避ける

## 危険パターン

- 海岸沿いの直進 → 遠距離視認
- 建物内の長居 → 第三者の漁夫の利
- AI交戦の長期化 → プレイヤーへの位置バレ
