## 記事の信頼度について

各記事には検証状況を示すメタ情報が付いています。

| 項目 | 意味 |
|---|---|
| **Status** | `published` = 公開可能と判断済み |
| **Confidence** | `low` = 仮説 / `medium` = 検証中・複数回確認 / `high` = 高信頼・再現性あり |
| **Updated** | 最終更新日 |

未確認の事項は本文中で「未確認」と明記しています。
