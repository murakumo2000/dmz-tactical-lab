# 運用ルール

## 1. チャット分離

- 戦術研究チャット：戦術・武器・マップ・ケースの検討
- システム設計チャット：Wiki構造・テンプレート・AI運用・公開設計

## 2. 記事ステータス

| status | 意味 |
|---|---|
| inbox | 未整理 |
| draft | 下書き |
| review | 検証中 |
| verified | 実プレイで有効性確認済み |
| published | 公開可能 |
| archived | 廃止・参考 |

## 3. 信頼度

| confidence | 意味 |
|---|---|
| low | 仮説 |
| medium | 複数回確認または妥当性あり |
| high | 実戦で再現性あり |

## 4. 命名規則

- 記事名：日本語
- タグ：英語
- frontmatterキー：英語
- 画像名：日本語可。ただし公開用は英数字推奨。

## 5. 公開判断

公開可能な記事だけ `80_Public` に配置する。
非公開の検証ログ、フレンド名、個別プレイ記録は `02_Private` に置く。
